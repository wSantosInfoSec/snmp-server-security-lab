# 🔍 Consulta SNMP Básica

## Objetivo
Realizar uma consulta SNMP ao roteador para validar o funcionamento
do serviço configurado.

## Ferramenta Utilizada
- MIB Browser

## Procedimento
- Endereço do agente SNMP: 192.168.1.1
- Versão SNMP: v2
- Community string (leitura): public
- OID consultado: Descrição do dispositivo

## Resultado
Foi possível obter informações do roteador a partir da MIB,
confirmando que o serviço SNMP está ativo e acessível.
