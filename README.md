# 🔐 SNMP Lab – Configuração Inicial

## 📌 Descrição
Este laboratório documenta a configuração básica de um servidor SNMP
em um ambiente controlado, com fins educacionais.

O objetivo inicial é compreender o funcionamento do SNMP e sua aplicação
em dispositivos de rede, servindo como base para análises de segurança futuras.

---

## 🎯 Objetivo
- Configurar o serviço SNMP em um roteador
- Validar conectividade entre host e dispositivo
- Realizar consultas básicas via SNMP

---

## 🛠️ Ferramentas Utilizadas
- Cisco Packet Tracer
- MIB Browser

---

## 🧪 Cenário
- Roteador: 192.168.1.1
- Host: 192.168.1.2
- SNMP v2 habilitado
- Community string configurada para leitura

---

## 🧩 Próximas Etapas
- Sniffing com Wireshark
- Análise de segurança da configuração SNMP
- Discussão sobre riscos de community strings padrão
- Boas práticas e hardening do serviço
